Installing plugins

Minecraft servers such as Spigot or Paper are able to run plugins which modify the experience within the server. Plugins come in many different forms and this guide explains how to setup and install plugins on your server.

Where can I find plugins?â€‹

You are able to find plugins published at many different places, some of these locations include SpigotMC and Hangar however there is many more sites available.

anteckning

Even though both SpigotMC and Hangar are reputable sites, malware is still occasionally uploaded. Please be cautious with what you are downloading and using.

Installing a pluginâ€‹

  1. You need to download the plugin file and after ensure it's built for the same Minecraft version that your server is running.
  2. Once obtaining the jar file you can upload it to your Minecraft server using either the panel file manager or a SFTP client like FileZilla.
  3. Upload the plugin file inside the directory named plugins
  4. After uploading the jar file you can go back to the console tab and restart the server.
